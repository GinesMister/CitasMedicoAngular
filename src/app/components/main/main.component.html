Sobre esta Aplicación

La aplicación de gestión de citas está diseñada para mejorar la organización y gestión de citas médicas. Sus principales funcionalidades incluyen:

  • Agendar citas para médicos y pacientes.
  • Editar información de médicos y pacientes.
  • Eliminar citas, médicos o pacientes según sea necesario.
